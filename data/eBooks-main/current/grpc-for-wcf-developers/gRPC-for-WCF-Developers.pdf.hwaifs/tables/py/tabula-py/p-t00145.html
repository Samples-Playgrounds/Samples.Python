Unnamed: 0 Metric type Description
0 Counter Tracks how often something happens, such as
1 requests and errors.
2 Gauge Records a single value that changes over time,
3 such as active connections.
4 Histogram Measures a distribution of values across
5 arbitrary limits. For example, a histogram can
6 track dataset size, counting how many
7 contained <10 records, how many contained
8 11-100 records, how many contained 101-1000
9 records, and how many contained >1000
10 records.
11 Meter Measures the rate at which an event occurs in
12 various time spans.
