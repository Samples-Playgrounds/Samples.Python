4 Vitess Vitess is a database solution for deploying,\nscaling, and managing large clusters of MySQL\ninstances. It can run in a public or private cloud\narchitecture. Vitess combines and extends many\nimportant MySQL features and features both\nvertical and horizontal sharding support.\nOriginated by YouTube, Vitess has been serving\nall YouTube database traffic since 2011.
