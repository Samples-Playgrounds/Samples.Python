|    | Method                        | Purpose                                                                    |
|---:|:------------------------------|:---------------------------------------------------------------------------|
|  0 | InitializeAsync               | Performs navigation to one of two pages when the app is launched.          |
|  1 | NavigateToAsync<T>            | Performs hierarchical navigation to a specified page.                      |
|  2 | NavigateToAsync<T>(parameter) | Performs hierarchical navigation to a specified page, passing a parameter. |
|  3 | RemoveLastFromBackStackAsync  | Removes the previous page from the navigation stack.                       |
|  4 | RemoveBackStackAsync          | Removes all the previous pages from the navigation stack.                  |
