Azure resource Event type Description
0 Azure subscription Microsoft.Resources.ResourceWriteSuccess Raised when a resource create or update operation succeeds.
1 None Microsoft.Resources.ResourceWriteFailure Raised when a resource create or update operation fails.

7 None Microsoft.Storage.BlobDeleted Raised when a blob is deleted.

10 None Microsoft.Devices.DeviceDeleted Published when a device is deleted from an IoT hub.

12 None Microsoft.Resources.ResourceWriteFailure Raised when a resource create or update operation fails.
