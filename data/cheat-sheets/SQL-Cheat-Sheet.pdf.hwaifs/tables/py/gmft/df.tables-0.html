Command Description Syntax Example
0 SELECT The SELECT command retrieves data from a database. SELECT column1, column2 FROM table_name; SELECT first_name, last_name FROM customers;
1 INSERT The INSERT command adds new records to a table. INSERT INTO table_name (column1, column2) VALUES (value1, value2); INSERT INTO customers (first_name, last_name) VALUES ('Mary', 'Doe');
2 UPDATE The UPDATE command is used to modify existing records in a table. UPDATE table_name SET column1 = value1, column2 = value2 WHERE condition; UPDATE employees SET employee_name = ‘John Doe’, department = ‘Marketing’;
3 DELETE The DELETE command removes records from a table. DELETE FROM table_name WHERE condition; DELETE FROM employees WHERE employee_name = ‘John Doe’;
