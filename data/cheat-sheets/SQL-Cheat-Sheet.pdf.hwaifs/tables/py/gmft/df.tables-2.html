MAX() MIN() AVG() SUM() COUNT() Aggregate Functions Command ALL ANY IN Command
0 The MAX command returns the maximum (highest) value in a specified column. The MIN command returns the minimum (lowest) value in a specified column. The AVG command is used to calculate the average (mean) of all values in a specified column. The SUM command is used to calculate the sum of all values in a specified column. The COUNT command counts the number of rows or non-null values in a specified column. Description Commands The ALL command is used to compare a value to all values returned by a subquery. It can be used with comparison operators like =, >, <, etc. The ANY command is used to compare a value to any value returned by a subquery. It can be used with comparison operators like =, >, <, etc. The IN command is used to determine whether a value matches any value in a subquery result. It is often used in the WHERE clause.
