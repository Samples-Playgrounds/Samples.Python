Command Description Syntax Example
0 CONCAT() The CONCAT command concatenates two or more strings into a single string. SELECT CONCAT(string1, string2,) AS concatenated_string FROM table_name; SELECT CONCAT(first_name, ' ', last_name) AS full_name FROM employees;
1 SUBSTRING()/SUBSTR() The SUBSTRING command extracts a substring from a string. SELECT SUBSTRING(string FROM start_position [FOR length]) AS substring FROM table_name; SELECT SUBSTRING(product_name FROM 1 FOR 5) AS substring FROM products;
2 CHAR_LENGTH()/LENGTH() The LENGTH command returns the length (number of characters) of a string. SELECT CHAR_LENGTH(string) AS length FROM table_name; SELECT CHAR_LENGTH(product_name) AS length FROM products;
3 UPPER() The UPPER command converts all characters in a string to uppercase. SELECT UPPER(string) AS uppercase_string FROM table_name; SELECT UPPER(first_name) AS uppercase_first_name FROM employees;
4 LOWER() The LOWER command converts all characters in a string to lowercase. SELECT LOWER(string) AS lowercase_string FROM table_name; SELECT LOWER(last_name) AS lowercase_last_name FROM employees;
5 TRIM() The TRIM command removes specified prefixes or suffixes (or whitespace by default) from a string. SELECT TRIM([LEADING | TRAILING | BOTH] characters FROM string) AS trimmed_string FROM table_name; SELECT TRIM(TRAILING ' ' FROM full_name) AS trimmed_full_name FROM customers;
6 LEFT() The LEFT command returns a specified number of characters from the left of a string. SELECT LEFT(string, num_characters) AS left_string FROM table_name; SELECT LEFT(product_name, 5) AS left_product_name FROM products;
7 RIGHT() The RIGHT command returns a specified number of characters from the right of a string. SELECT RIGHT(string, num_characters) AS right_string FROM table_name; SELECT RIGHT(order_number, 4) AS right_order_number FROM orders;
8 REPLACE() The REPLACE command replaces occurrences of a substring within a string. SELECT REPLACE(string, old_substring, new_substring) AS replaced_string FROM table_name; SELECT REPLACE(description, 'old_string', 'new_string') AS replaced_description FROM product_descriptions;
