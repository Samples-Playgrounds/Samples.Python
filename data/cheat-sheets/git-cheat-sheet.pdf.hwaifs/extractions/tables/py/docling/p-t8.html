0 git log [-n count] List commit history of current branch. -n count limits list to last n commits.
1 git log --oneline --graph --decorate An overview with reference labels and history graph. One commit per line.
2 git log ref-. List commits that are present on the current branch and not merged into ref . A ref can be a branch name or a tag name.
3 git log -.ref List commit that are present on ref and not merged into current branch.
4 git reflog List operations (e.g. checkouts or commits) made on local repository.
