0 git fetch [remote] Fetch changes from the remote , but not update tracking branches.
1 git fetch --prune [remote] Delete remote Refs that were removed from the remote repository.
2 git pull [remote] Fetch changes from the remote and merge current branch with its upstream.
3 git push [--tags] [remote] Push local changes to the remote . Use --tags to push tags.
4 git push -u [remote] [branch] Push local branch to remote repository. Set its copy as an upstream.
