Navigating Directories
1s List the contents of the directory. ls -larth /home/mcuser/science/code
cd Change directory. cd /home/mcuser/science
pwd present working directory. pwd
* Wild card, any number of chars. ls *.txt
? Wild card, 0 or 1 characters. ls foo?.txt
Current directory. cd .
Directory one level up. cd ..
~ Home directory. cd ~
- Previous directory. cd -

Modifying Directories and Files

mkdir Make new empty directory.

rmdir Remove empty directory.

mv Move files (overwrites destination).

cp Copy files (overwrites destination).

rm Remove files. CANNOT UNDO!
Change file permissions.

chmod rwx = read, write, execute.

ugoa = user, group, other, all.
