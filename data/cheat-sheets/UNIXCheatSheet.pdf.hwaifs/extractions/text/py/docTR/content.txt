pwd

Modifying Directories and Files
mkdir
Make new empty directory.
mkdir MyNewDirectory
rmdir
Remove empty directory.
rmdir MyNewDirectory
mv
Move files (overwrites destination). mv -i foo.txt foo2.txt
cp
Copy files (overwrites destination). cp -i ../foo.txt
rm
Remove files. CANNOT UNDO! rm -i foo.txt
Change file permissions.
chmod a+rwx badger txt
chmod
rwx = read, write, execute.
chmod O-WX badger.txt
ugoa i user, group, other, all.
chmod u+x myscript . sh
Viewing Files in Terminal
less
View files in terminal window.
less foo.txt
cat
Concatenate files.
cat foo1.txt foo2.txt
head
Display first 10 lines.
head -20 foo.txt
tail
Display last 10 lines.
tail -f logfile.txt
Searching
grep myvarname *.txt
grep
Search for word(s) in a file.
grep rodent *.txt
grep "Hello World" *.txt
find playground -name "*.txt"
find
Search a path for files /directories find playground -type d
that match some specification.
find playground -not -name "*.txt' "

WC
Word count.

Sort input alphanumerically.
cat foo.txt I I less

sort myfile.txt I grep mykeyword
program.
>
Redirect standard output to. Over- cat foo.txt > foo2.txt
writes existing file.
>>
Redirect standard output and con- cat foo.txt >> foo2.txt
catenate to.
<
Redirect standard input from.
less < foo.txt

Advanced operations by column.

Advanced operations by row.
Job process Manipulation
&
Run process in background.
/run_job.sh &
jobs
Show your current processes
jobs
ps
Show processes more flexible)
ps -u username
top
Show top active processes
top -u username
kill
Stop a job
kill processid
Ctrl-z
Pause current job
/run_job.sh; Ctrl-z
Ctrl-c
Cancel current job
/run_job.sh; Ctrl-c
bg
Resume process in background
/run_job.s sh; Ctrl-z; bg %1
fg
Bring process to foreground
fg %2
Remote Connections
ssh
Secure shell login.
ssh -Y user@host.edu
scp
Secure copy.
scp user@host.edu path/to/file.txt
wget
Download file from remote URL.
wget www.google.com -0 google.html
Ctrl-d
Exit login. Sometimes disabled.
Can also use exit, logout
