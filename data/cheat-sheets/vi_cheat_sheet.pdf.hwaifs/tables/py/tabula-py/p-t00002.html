Deletion commands Unnamed: 0 Move text from file old to file new
0 dd  or ndd Delete n lines to general buffer vi old
1 dw Delete word to general buffer “a10yy yank 10 lines to buffer a
2 dnw Delete n words :w write work buffer
3 d) Delete to end of sentence :e new edit new file
4 db Delete previous word “ap put text from a after cursor
5 D Delete to end of line :30,60w new Write lines 30 to 60 in file new
6 x Delete character
7 Regular expressions (search strings)
8 Recovering deletions
9 ^ Matches beginning of line
10 p Put general buffer after cursor $ Matches end of line
11 P Put general buffer before cursor . Matches any single character
12 * Matches any previous character
13 Undo commands .* Matches any character
14 u Undo last change Search and replace commands
15 U Undo all changes on line
16 Syntax:
17 Rearrangement commands
18 :[address]s/old_text/new_text/
19 yy or Y Yank (copy) line to general buffer
20 “z6yy Yank 6 lines to buffer z Address components:
21 yw Yank word to general buffer . Current line
22 “a9dd Delete 9 lines to buffer a n Line number n
23 “A9dd Delete 9 lines; Append to buffer a .+m Current line plus m lines
24 “ap Put text from buffer a  after cursor $ Last line
25 p Put general buffer after cursor /string/ A line that contains "string"
26 P Put general buffer before cursor % Entire file
27 J Join lines [addr1],[addr2] Specifies a range
