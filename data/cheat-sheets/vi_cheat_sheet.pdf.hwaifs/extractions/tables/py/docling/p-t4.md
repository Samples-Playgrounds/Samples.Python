|    | File management   | commands                         |
|---:|:------------------|:---------------------------------|
|  0 | :w name           | Write edit buffer to file name   |
|  1 | :wq               | Write to file and quit           |
|  2 | :q!               | Quit without saving changes      |
|  3 | ZZ                | Same as :wq                      |
|  4 | :sh               | Execute shell commands (<ctrl>d) |
