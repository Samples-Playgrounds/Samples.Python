|    | 0                            | 1                                           |
|---:|:-----------------------------|:--------------------------------------------|
|  0 | Alt+Click                    | Insert cursor                               |
|  1 | Ctrl+Alt+ ↑ / ↓              | Insert cursor above / below                 |
|  2 | Ctrl+U                       | Undo last cursor operation                  |
|  3 | Shift+Alt+I                  | Insert cursor at end of each line selected  |
|  4 | Ctrl+L                       | Select current line                         |
|  5 | Ctrl+Shift+L                 | Select all occurrences of current selection |
|  6 | Ctrl+F2                      | Select all occurrences of current word      |
|  7 | Shift+Alt+ →                 | Expand selection                            |
|  8 | Shift+Alt+ ←                 | Shrink selection                            |
|  9 | Shift+Alt + (drag mouse)     | Column (box) selection                      |
| 10 | Ctrl+Shift+Alt + (arrow key) | Column (box) selection                      |
| 11 | Ctrl+Shift+Alt +PgUp/PgDn    | Column (box) selection page up/down         |
