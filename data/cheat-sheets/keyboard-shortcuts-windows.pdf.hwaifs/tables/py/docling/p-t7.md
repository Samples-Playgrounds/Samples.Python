|---:|:-----------------------|:--------------------------------------------|
|  0 | Ctrl+F4, Ctrl+W        | Close editor                                |
|  1 | Ctrl+K F               | Close folder                                |
|  2 | Ctrl+\                 | Split editor                                |
|  3 | Ctrl+ 1 / 2 / 3        | Focus into 1 st , 2 nd or 3 rd editor group |
|  4 | Ctrl+K Ctrl+ ←/→       | Focus into previous/next editor group       |
|  5 | Ctrl+Shift+PgUp / PgDn | Move editor left/right                      |
|  6 | Ctrl+K ←/→             | Move active editor group                    |
